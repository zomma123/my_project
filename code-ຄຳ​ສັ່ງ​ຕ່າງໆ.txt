ຄຳ​ສັ່ງ laravel + vue

- composer install / composer i     : ເປັນ​ຄຳ​ສັ່ງ​ຕິດ​ຕັ້ງ libary ຂອງ laravel
- npm install / npm i               : ຄຳ​ສັ່ງຕິດ​ຕັ້ງ nodejs
- php artisan serve                 : ຄຳ​ສັ່ງ run serve ຂອງ laravel
- npm run dev                       : ຄຳ​ສັ່ງ run compli vue
- npm run watch                     : ຄຳ​ສັ່ງ run ຕິດ​ຕາມ ແລະ compli vue

ຄຳ​ສັ່ງ​ຂອງ git 

- git add . / git add ຊື່​ໄຟ​ລ໌        : ເປັນ​ການ​ເພີ່ມ​ໄຟ​ລ໌​ເຂົ້າ​ໄປ​ໃນ git
- git commit -m "ຄຳ​ອະ​ທິ​ບາຍ"        : ​ເປັນ​ການ commit ບັນ​ທຶກ code
- git push                        : ເປັນ​ການ​ ອັດ​ໂຫຼດ​ code ຂຶ້ນ github
- git pull                        : ເປັນ​ການ ອັດ​ເດດ code ລ່າ​ສຸດ​ຈາກ github
- git add . && git commit -m "ຄຳ​ອະ​ທິ​ບາຍ" && git push : ວິ​ທີ​ຂຽນ​ລວມ​ກັນເປັນ​ແຖວດຽວ
